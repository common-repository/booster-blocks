Booster Blocks is a collection of new and exciting blocks to enhance usability of WordPress blocks. 

== Description ==

Booster Blocks add new powerful blocks to your WordPress's default block collection. These new unique blocks allow you to design website layouts with less time and more creativity. All the blocks have extensive styling capabilities to match your theme's look and feel. 
With an easy to use interface and several customisations, you can easily create stunning layouts of your choice.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= Will it slow down my site speed? =

Booster Blocks is a lightweight plugin, optimized for fast loading.

= Do I need to Configure the plugin? =

No configurations are required to use this plugin. Once you install and activate the plugin, you are ready to go.

= Where can I find the new block added? =

New blocks are added at the end of the default block collection area under the category BOOSTER BLOCKS”.
